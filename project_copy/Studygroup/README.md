# StudySync - AI-Powered Group Study & Learning Hub

## 🎓 Overview
StudySync is an innovative ed-tech platform that combines real-time collaboration with AI-powered learning tools. It enables students to create virtual study rooms, collaborate through video calls, share notes, and get AI assistance for their studies.

## 🚀 Features
- Real-time video conferencing with screen sharing
- Collaborative whiteboard
- AI-powered study assistant
- Question bank generator
- Shared notes and resources
- Book scanning and OCR capabilities

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Python (Flask)
- **Database**: SQLite (for development)
- **AI Integration**: OpenAI API
- **Video Calls**: WebRTC
- **Authentication**: JWT

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Modern web browser
- Node.js (for development tools)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/studysync.git
cd studysync
```

2. Set up the backend:
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. Start the backend server:
```bash
python app.py
```

4. Open the frontend:
- Navigate to the `frontend` folder
- Open `index.html` in your web browser

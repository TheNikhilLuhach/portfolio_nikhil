Hand Gesture Control System

A computer vision application that enables touchless computer control through hand gestures using MediaPipe for real-time hand tracking.

Python OpenCV MediaPipe Computer Vision

🤚 Gesture Commands

☝️
One Finger
Opens Notepad for quick note-taking
✌️
Two Fingers
Opens CMD and launches Python interpreter
🤟
Three Fingers
Opens new CMD and launches Jupyter Notebook
✊
Fist
Minimizes all windows (Win+D)

🎯 Accuracy Features

  • Adjustable sensitivity settings

🛡️ Safety Features

  • Automatic cleanup on exit
  • Hand tracking boundary detection
  • Error handling for failed launches
  • Keyboard shortcut override (Q to quit)
